# FireCalc — Retirement Planner

A modern, interactive retirement planning calculator built with React, Vite, Tailwind CSS, and Recharts.

## Features

- **Real-time calculations** — all charts and numbers update instantly as you move sliders
- **4 chart views** — portfolio over time, savings needed by retirement age, max spendable income, and nest egg required
- **Monte Carlo mode** — set uncertainty ranges for return rate, inflation, income growth, and spending variability; run thousands of simulations to see percentile outcome bands
- **Coast FIRE** — calculates the age you can stop saving and let compounding do the rest
- **Inflation-adjusted math** — all spending targets grow with inflation automatically

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build & Deploy

```bash
npm run build
```

Output goes to `dist/`. Deploy to Vercel by connecting your GitHub repo — Vercel auto-detects Vite and sets the build command and output directory correctly.
